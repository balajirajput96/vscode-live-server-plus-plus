# 🚀 Master Prompt System - Balaji Nonprofit + Automation

## 📋 Overview

The Master Prompt System is a comprehensive AI-powered solution designed specifically for Balaji's nonprofit/educational initiatives at Parul University. This system provides ready-to-use prompts, automation workflows, and templates for maximizing free/discount access to AI tools while setting up error-free automations.

## 🎯 Key Features

### 1. **Universal Master Prompt**
- 🔥 **Featured Prompt**: Works with any AI Pro/Enterprise tool
- 🎛️ **Customizable Inputs**: Tailor for specific questions and platforms
- 📋 **Structured Output**: 10 specific sections including summaries, action plans, and checklists
- 🔒 **ToS Compliant**: Built with legal compliance and security best practices

### 2. **n8n Automation Workflows**
- 📤 **Donor Management**: Automated donor form → AI reply → Drive save → Teams alert
- 🔧 **Error-Safe Design**: Clear placeholders for easy customization
- 📊 **Integration Ready**: Microsoft Teams, Google Sheets, OpenAI compatible

### 3. **Professional Email Templates**
- 🎓 **Nonprofit Benefits**: Organization verification requests
- 🔬 **Research Credits**: Academic program applications  
- 💰 **Grant Applications**: Funding inquiry templates
- 📝 **Ready-to-Use**: Just customize with your details

### 4. **API/CLI Snippets**
- 🤖 **OpenAI Integration**: Secure authentication and usage
- 🧠 **Google Gemini**: API setup with best practices
- 🤗 **Hugging Face**: Token-based authentication
- 🔒 **Security First**: Environment variables and error handling

## 🚀 Quick Start

### Step 1: Access the Master Prompt
1. Open the Career Automation System
2. Navigate to "AI Prompts" tab
3. Find the **⭐ FEATURED** "मास्टर प्रॉम्प्ट (Balaji Nonprofit + Automation)" card

### Step 2: Customize Your Prompt
1. Click **"Customize"** button
2. Fill in your specific details:
   - **Latest Question/Task**: Your current requirement
   - **Screenshots/Links**: Any relevant documentation
   - **Priority Platforms**: Tools you want to focus on
   - **Constraints**: Budget, legal, or technical limitations

### Step 3: Copy and Use
1. Click **"Copy Master Prompt"**
2. Paste into any AI Pro/Enterprise tool (ChatGPT Plus, Claude Pro, etc.)
3. Get structured, actionable outputs

## 📊 Master Prompt Output Structure

When you use the Master Prompt, you'll receive:

1. **Summary** (2-3 lines): What to do first with rationale
2. **Programs/Offers**: Free/discounted tools with eligibility criteria
3. **Action Plan**: Step-by-step instructions with exact button paths
4. **Risk Notes**: ToS compliance warnings and safe alternatives
5. **Ready Emails**: 3 professional email templates
6. **n8n Pack**: Importable workflow JSONs
7. **API/CLI Snippets**: Code with security best practices
8. **Contingency**: Open-source alternatives
9. **One-Page Checklist**: Immediate/This Week/Optional tasks
10. **Analysis**: Integration of your screenshots/links

## 🆘 Troubleshooting

### Common Issues:

**Issue**: Clipboard copy not working
**Solution**: Use Ctrl+A to select all text manually

**Issue**: n8n import fails
**Solution**: Check JSON format and node compatibility

**Issue**: Email templates get spam filtered
**Solution**: Use institutional email and proper subject lines

**Issue**: API snippets not working
**Solution**: Verify environment variables and API key validity
