# 🚀 Quick Start Guide - AI Career Automation Dashboard

## ⚡ तुरंत शुरू करने के लिए

### Windows Users
```bash
# 1. डबल-क्लिक करें
start-dashboard.bat

# या Command Prompt में
start-dashboard.bat
```

### Linux/Mac Users
```bash
# 1. Terminal खोलें
# 2. फोल्डर में जाएं
cd career-automation-system

# 3. Script चलाएं
./start-dashboard.sh

# या Python सीधे चलाएं
python3 server.py
```

### Manual Start (सभी OS)
```bash
# 1. Python सर्वर शुरू करें
python -m http.server 8000

# 2. ब्राउज़र में खोलें
http://localhost:8000
```

## 🎯 पहले 5 मिनट में क्या करें

### 1. Portfolio Builder (2 मिनट)
1. **Portfolio Builder** टैब पर क्लिक करें
2. अपने पहले प्रोजेक्ट की जानकारी भरें:
   - **Project Name**: Gene Expression Analysis
   - **Description**: Bioinformatics project using Python
   - **Tools Used**: Python, Pandas, Matplotlib
   - **Dataset Source**: Kaggle/NCBI
   - **Key Findings**: Found interesting patterns
3. **"AI से README बनाएं"** बटन पर क्लिक करें
4. जनरेटेड README को कॉपी करें

### 2. LinkedIn Profile (2 मिनट)
1. **Resume Optimizer** टैब पर जाएं
2. अपनी जानकारी भरें:
   - **Experience**: 1 वर्ष
   - **Target Role**: Bioinformatics Analyst
   - **Key Skills**: Python, SQL, Data Analysis
3. **"LinkedIn प्रोफाइल बनाएं"** बटन पर क्लिक करें
4. जनरेटेड हेडलाइन और About सेक्शन को कॉपी करें

### 3. Social Media Post (1 मिनट)
1. **Social Media Generator** टैब पर जाएं
2. अपने प्रोजेक्ट के लिए पोस्ट बनाएं
3. जनरेटेड LinkedIn पोस्ट को कॉपी करें

## 📱 तुरंत उपयोग करने योग्य AI Prompts

### Wix Website Builder के लिए
```
Create a professional portfolio website for a biotechnology professional transitioning into bioinformatics and data analysis featuring skills in Python, SQL, web design, and digital marketing. Use modern, clean design in blue, white, grey. Add pages: Home, About Me, Skills, Projects, Blog, Contact.
```

### LinkedIn Headline के लिए
```
Act as a professional career coach. Write 5 powerful and professional LinkedIn headlines. I have Diploma in Biotechnology, skills in Python, web design, digital marketing, seeking bioinformatics/data analysis role in pharmaceutical industry.
```

### LinkedIn About Section के लिए
```
Act as a professional resume writer. Write a compelling About section (~150-200 words) with my key details: Diploma in Biotechnology, 1‑month internship in bioinformatics, skills Python, SQL, web design, digital marketing; and goal to work in pharmaceutical or clinical research companies.
```

### Social Media Post के लिए
```
Act as a social media marketing expert for biotech. Create an engaging LinkedIn post about my portfolio project: [Project Name], Goal: [Description], Tools Used: Python, Pandas, Matplotlib, Key Finding: [Insight], Call to Action: Visit my portfolio website. Include: compelling hook, simple explanation, highlight result, skills mention, CTA, 5–7 hashtags (#Bioinformatics #DataAnalysis #Biotechnology #Python #Pharma #ClinicalResearch).
```

## 🔧 Troubleshooting

### Dashboard नहीं खुल रहा है?
```bash
# 1. Python चेक करें
python --version

# 2. Port चेck करें
netstat -an | grep 8000

# 3. अलग port पर चलाएं
python -m http.server 8080
```

### Copy Function काम नहीं कर रहा?
- HTTPS connection चेक करें
- Browser permissions चेक करें
- Manual copy करें (Ctrl+C)

### AI Content Generate नहीं हो रहा?
- Internet connection चेक करें
- Browser refresh करें
- Console में errors देखें (F12)
