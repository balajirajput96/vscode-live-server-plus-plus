## 🎯 AI Prompts Library

### 📱 Social Media Content Prompts

#### LinkedIn Post Generator
```
"Create a professional LinkedIn post about [TOPIC] for a biotechnology professional. Include:
- Hook in first line
- 3-4 key points
- Call to action
- Relevant hashtags (max 5)
- Professional tone with personality"
```

#### Facebook Post Generator
```
"Write an engaging Facebook post about [TOPIC] that:
- Starts with an attention-grabbing question
- Shares personal insight or experience
- Encourages comments and engagement
- Uses emojis appropriately
- Includes relevant hashtags"
```

### 🛠️ Technical Content Prompts

#### Project Description Generator
```
"Create a professional project description for [PROJECT NAME]:
- Project overview and objectives
- Technologies and tools used
- Methodology and approach
- Key results and achievements
- Impact and applications
- Future improvements
Keep it concise and technical but accessible."
```

#### README Generator
```
"Generate a comprehensive README.md for a [PROJECT TYPE] project:
- Clear project title and description
- Installation instructions
- Usage examples with code
- API documentation (if applicable)
- Contributing guidelines
- License information
- Contact details"
```

### 💼 Career Content Prompts

#### Resume Bullet Points
```
"Transform this job responsibility into an impactful resume bullet point:
[INSERT RESPONSIBILITY]

Make it:
- Action-oriented (start with strong verb)
- Quantified with metrics when possible
- Results-focused
- ATS-friendly
- Professional tone"
```

#### Cover Letter Generator
```
"Write a compelling cover letter paragraph for [POSITION] at [COMPANY]:
- Show knowledge of company/role
- Highlight relevant experience
- Demonstrate value proposition
- Express genuine interest
- Professional but engaging tone"
```
