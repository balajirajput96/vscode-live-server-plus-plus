### Step 1: Set Up Automation Agent (NEW)

```bash

# Configure environment

# Edit .env with your API keys

# Test automation services
npm run automation:jobs      # Job aggregation
npm run automation:learning  # Learning tracker
```

## 🔧 Troubleshooting (Enhanced)

### Common Issues

**Issue**: Automation not running
**Solution**: Check environment variables and API credentials

**Issue**: WhatsApp not receiving messages
**Solution**: Verify WhatsApp Business API setup and phone number verification

**Issue**: Google Sheets access denied
**Solution**: Check service account permissions and API key configuration

**Issue**: CI failures not being classified
**Solution**: Verify GitHub token permissions and workflow configuration

**Issue**: Job scraping not working
**Solution**: Job sites may have anti-scraping measures; try different sources
