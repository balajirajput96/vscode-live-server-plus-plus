## 🚀 Quick Start Guide

### 1. Setup Environment
```bash
# Run quick setup
python3 scripts/quick-setup.py

# Configure credentials
cp .env.example .env
# Edit .env with your actual credentials
```

### 2. Start Services
```bash
# Development mode
npm run n8n:start

# Verify setup
npm run n8n:setup
```

### 3. Test Workflow
```bash
# Run comprehensive tests
npm run n8n:test

# View demo functionality
npm run n8n:demo
```

### 4. Import Workflow
1. Access n8n at `http://localhost:5678`
2. Import `workflows/balaji-automation-workflow.json`
3. Configure credentials for each service
4. Test and activate the workflow
