Welcome to ML Deployment Platform! 🚀

A complete production-ready platform for deploying and managing machine learning models with real-time monitoring and analytics.

✨ Features

  • 📤 Model Upload - Upload and deploy ML models (.pkl, .joblib)
  • 🎯 Single Predictions - Make individual predictions with confidence scores
  • 📦 Batch Processing - Process multiple predictions efficiently
  • 📊 Real-time Monitoring - Track performance metrics and statistics
  • 🔬 API Testing - Interactive API documentation and testing
  • 📈 Analytics Dashboard - Visualize model usage and predictions

🚀 Quick Start

  1. Upload your trained ML model using the "Upload Model" tab
  2. Test predictions using the "Predict" tab
  3. Process batch data using the "Batch Predict" tab
  4. Monitor performance in the "Monitoring" tab
  5. Access API endpoints via the "API Docs" tab

API Documentation

POST /models/upload

Upload and deploy a new ML model

curl -X POST http://localhost:8000/models/upload \ -F "file=@model.pkl" \ -F "name=My Model" \ -F "version=1.0"

GET /models

List all deployed models

curl http://localhost:8000/models

POST /predict

Make a single prediction

curl -X POST http://localhost:8000/predict \ -H "Content-Type: application/json" \ -d '{ "model_id": "model-id-here", "features": [5.1, 3.5, 1.4, 0.2] }'

POST /predict/batch

Make batch predictions

curl -X POST http://localhost:8000/predict/batch \ -H "Content-Type: application/json" \ -d '{ "model_id": "model-id-here", "data": [[5.1, 3.5, 1.4, 0.2], [4.9, 3.0, 1.4, 0.2]] }'

GET /stats

Get platform statistics

curl http://localhost:8000/stats

GET /health

Health check endpoint

curl http://localhost:8000/health
