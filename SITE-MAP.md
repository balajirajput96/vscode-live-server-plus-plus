## 🎨 Customization

### Update Content
1. Edit HTML files in repository
2. Commit and push changes
3. GitHub Actions auto-deploys
4. Changes live in 1-2 minutes

### Add New Pages
1. Create new HTML file in root
2. Add links from index.html
3. Push to repository
4. New page auto-deployed

### Modify Styling
1. Edit inline CSS in HTML files
2. Or create external CSS files
3. Link from HTML files
4. Push and deploy
