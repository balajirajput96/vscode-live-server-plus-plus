## 🚨 Troubleshooting

### Site Not Loading?
- **Wait:** Initial deployment takes 1-2 minutes
- **Check Actions:** Verify workflow completed successfully
- **Clear Cache:** Try incognito/private mode
- **Manual Trigger:** Push an empty commit to trigger rebuild

### 404 Errors?
- **Verify Source:** Ensure source is set to "GitHub Actions" or "main branch"
- **Check Files:** Confirm index.html exists in root directory
- **Wait:** Give it 2-3 minutes for DNS propagation

### Workflow Failed?
- **Check Permissions:** Ensure Pages is enabled in repository settings
- **Review Logs:** Click on failed workflow to see error details
- **Re-run:** Click "Re-run all jobs" button
